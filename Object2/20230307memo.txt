오버라이딩

조상 클래스로부터 상속받은 메서드의 내용을 변경하는 것

ex)

class Point {
    int x;
    int y;

    String getLocation() {
        return "x :" + x + ", y :"+y;
    }
}

class Point3D extends Point {
    int z;

    String getLocation() {      // 오버라이딩
        return "x :" + x + ", y :" + y + ", z :" + z;
}

오버라이딩의 조건!

오버라이딩은 메서드의 '내용'만을 새로 작성하는 것. 따라서 메서드의 선언부는 조상의 것과 일치해야 함.

이름, 매개변수, 반환타입도 같아야함.(선언부가 서로 일치해야함)
